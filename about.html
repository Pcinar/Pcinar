This Ivy Tech Web Development Site

My name is Poyraz Cinar. I built this website for an Ivy Tech web development assignment, focusing on usability, accessibility, basic SEO and a little bit about myself.

How I kept the site accessible

  • Readable font size (16px+)
  • Clear heading structure (H1 → H2)
  • Consistent navigation on every page
  • Alt text provided for images

How I improved search visibility

  • Keyword used in titles and headings
  • Meta descriptions on every page
  • Internal links between pages
  • Descriptive filenames for images
